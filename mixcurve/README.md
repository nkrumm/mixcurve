
## Mixcurve (a python package)

Mixcurve is designed to interpret 1:1 mixing data from the coagulation lab. The data is of the form:


|         | PT | PTT | TT | Fibrinogen |
|---------|----|-----|----|------------|
| Patient | 14 |  40 | 23 |        400 |
| Mix     | 14 |  32 |    |            |
| Pool    | 13 |  29 |    |            |
